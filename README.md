# One Package

看到 [gopkg](https://github.com/bytedance/gopkg/tree/main) 项目后, 感觉一股啰嗦妈宝味道. 为了方便自己提炼内在细节, 构造这个项目用于学习和工程使用.

## How To Use

You can use `go get -u github.com/wangzhione/onepkg@latest` to get or update `onepkg`
